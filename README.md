## Task 2
Install ffmpeg into the system.

**Windows**: Download from official ffmpeg website (https://www.ffmpeg.org/), Add ffmpeg into system path \
**Ubuntu/Linux**: ``` sudo apt install ffmpeg ``` \
**MacOS**: ``` brew install ffmpeg ```
